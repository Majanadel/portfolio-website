FitMe

  • Role: UI Designer
  • Tools used: Sketch, InVision, Marvel, Adobe Illustrator
  • Timeline: 6 Months

Objective

FitMe is an exercise app created for individuals with busy schedules. It helps people get on track with exercise and create healthy routines for the future.

Problem Statement

If you have been out of the loop for a while, it’s tough to get back in any activity, especially exercise. Starting new routines with a schedule that doesn’t allow any extras can be very challenging.

Possible Solutions

FitMe will help busy professionals create workout routines that can fit into their schedules by providing easy scheduling, exercise guides for beginners, keep a track of their progress and much more.

User Flow

I created a user flow to follow the steps that my user will have to go through in order to achieve their goals.

Low Fidelity Wireframes

In Low-fidelity diagrams, I wanted to create basic details and layout of the screens for FitMe and test each screen's basic functionality in order for my users to achieve their goals.

Mid Fidelity Wireframes

I created mid-fidelity wireframes using Sketch. I developed some of the UX patterns for my wireframes such as input feedback, password strength, hamburger menu and modal window, pagination and pickers, etc.

Moodboards

User persona Rebecca wants to be challenged and get back on the track with her exercise, so I used orange color in the first mood board which represents fun, movement, competition, confidence, impulse, haste, etc. In the second mood board I used black color which represents elegance, value, sophistication, power. This would go good with Rebecca’s lifestyle as a young tech professional.

Style Guide

I created typography, chose colors, imagery, animation, logo and icons and other UI elements.

High Fidelity Wireframes

Design was slowly coming together in High fidelity wireframes and now I’m able to see form and function of the app. I used Sketch to create wireframes.

Design for Different Breakpoints

Using a 12-column grid, I created design for mobile, iPad and Desktop

Final Product

App that it’s not overwhelming for a user. Contains scheduling system and notifications to remind user of this new routine in creation. Exercises have videos and clear step-by-step instructions. It’s easy to track progress. This app will fit in the busiest schedules.

Takeaways

This was my first UI project. I learned how to create my own icons, logo, mood boards. And also, how to create responsive design for different breakpoints.
